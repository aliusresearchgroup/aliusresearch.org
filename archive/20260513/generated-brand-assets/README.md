# Generated Brand Assets

This archive batch contains two standalone animated vector reconstructions of the ALIUS leaf logo.

- `alius-logo-animated-vector-gentle-wind.svg`: the calmer animated version, using three overlapping vector leaf paths.
- `alius-logo-animated-vector-strong-wind.svg`: the faster, wider-swaying strong-wind version.

Both files keep the original `viewBox="0 0 1023 1111"` leaf-logo coordinate system and generate overlap tones with SVG multiply blending. They do not embed PNG/GIF frame data.
