{\LARGE\bfseries ALIUS Bulletin Editor Checklist\par}
\vspace{1em}
\begin{itemize}[leftmargin=1.5em]
\item Confirm the piece subtype: interview, conversation, podcast, commentary, review, or editorial note.
\item Keep the published bulletin look as the visual target, not the working Word-template look.
\item Provide a title, abstract, keywords, contributor block, citation block, and running title.
\item Keep pull quotes explicit and editor-approved. For interviews and conversations, aim for 2--6 excerpts.
\item Use American-English punctuation conventions, straight ALIUS spacing rules, and APA references.
\item Check front-matter wording carefully because historical issues differ in small human-edited ways.
\item Run the local preflight and validation scripts before publishing a new standardized PDF.
\end{itemize}
