# ALIUS Brand Logo Assets

This public folder keeps only the current website logo assets. Historical SVG variants have been moved out of the published site tree so they are no longer served from /assets/brand/.

- `alius-logo.svg`: current live website logo, with all-leaf mold sprouting, out-of-phase leaf sway, and a per-letter liquid-mould Century Gothic wordmark reveal.
- `alius-logo.pdf`: static logo PDF exported from the recreated vector logo.
- `alius-logo-font-info.pdf`: font identification and layout notes. The typeface is Century Gothic Regular.
- `originals/1477332210-original-weebly-logo.png`: original exported website raster logo.

Repository-only SVG archives:

- `../../../../archive/20260520/logo-svg-archive/`: historical SVG logo variants removed from public website access on 2026-05-20.
- `../../../../archive/20260519/generated-brand-assets/alius-logo-animated-vector-final-water-mould-out-of-phase-sway-20260519.svg`: archived copy of the final preferred SVG.
