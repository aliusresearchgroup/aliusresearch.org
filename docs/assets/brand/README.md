# ALIUS Brand Logo Archive

This folder preserves the logo materials used for the website header.

- `alius-logo.svg`: live website logo, with the animated vector leaf and the recreated Century Gothic text layout.
- `alius-logo-animated.svg`: archived copy of the animated full logo.
- `alius-leaf-animated.svg`: animated leaf-only SVG component.
- `alius-logo-static.svg`: archived static SVG recreation of the original raster logo.
- `alius-logo.pdf`: static logo PDF exported from the recreated vector logo.
- `alius-logo-font-info.pdf`: font identification and layout notes. The typeface is Century Gothic Regular.
- `originals/1477332210-original-weebly-logo.png`: original exported website raster logo.
- `originals/logo-ALIUS-original-leaf.svg`: original supplied static leaf SVG.
- `originals/logo-ALIUS-original-animated-leaf.svg`: original supplied animated leaf SVG.
